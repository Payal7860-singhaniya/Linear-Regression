# 📊 Linear Regression Learning App

An interactive **Streamlit application** to learn and implement **Linear Regression** using your own dataset.

---

## 🚀 Features

- Upload CSV dataset  
- Data preprocessing (handle missing values & normalization)  
- Scatter plot & correlation matrix  
- Train Linear Regression model  
- Regression visualization  
- Make predictions  
- Evaluate model (MSE, MAE, R² Score)  
- Residual plot  
- Download processed dataset  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## ⚙️ Setup & Run

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
pip install streamlit pandas numpy matplotlib scikit-learn
streamlit run app.py
